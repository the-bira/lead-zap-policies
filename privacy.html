Política de Privacidade

LeadZap

1. Introdução

Esta Política de Privacidade descreve como o LeadZap coleta, utiliza, armazena e protege as informações dos usuários – tanto dos donos de negócios quanto dos clientes finais – que utilizam nossa plataforma para gerenciamento de leads, agendamentos e comunicação via WhatsApp com chatbot de IA.

2. Dados Coletados

2.1. Dados de Usuários (Donos de Negócio)

  • Dados cadastrais: Nome, e-mail, senha (armazenada de forma criptografada) e imagem do perfil.
  • Dados de autenticação: Informações provenientes do Active Directory (quando aplicável), registros de login e dados de sessão.
  • Dados operacionais: Configurações de conta, conexões com o WhatsApp, tokens e identificadores (como phone_number_id) obtidos via API do Meta, dados de agendamentos e histórico de conversas.

2.2. Dados dos Clientes Finais (Leads)

  • Contato: Número de telefone, nome (quando fornecido) e dados coletados durante a interação com o chatbot (respostas, lead score, etc.).
  • Histórico de Conversa: Mensagens enviadas e recebidas, registros de agendamentos e interações registradas na plataforma.

3. Como Utilizamos os Dados

  • Para prestar os serviços: Viabilizar a comunicação via WhatsApp, qualificar leads e agendar serviços, além de exibir históricos e métricas no dashboard.
  • Melhoria e Personalização: Otimizar a experiência do usuário e personalizar o atendimento.
  • Segurança: Integrar com sistemas de segurança, como Active Directory e softwares de endpoint, para proteger a infraestrutura e os dados.

4. Compartilhamento e Armazenamento dos Dados

  • Os dados são armazenados em bancos de dados gerenciados e protegidos por medidas de segurança.
  • Os dados só são compartilhados com terceiros quando necessário para a prestação dos serviços ou para cumprimento de obrigações legais.

5. Direitos dos Usuários

  • Acesso, correção e exclusão dos dados pessoais mediante solicitação.
  • Revogação de consentimento, sabendo que isso pode impactar a prestação dos serviços.

6. Segurança dos Dados

  • Implementação de medidas técnicas e administrativas para proteção dos dados.
  • Uso de Active Directory para gerenciamento centralizado e aplicação de políticas de segurança.

7. Alterações na Política

Esta política pode ser atualizada periodicamente. Quaisquer alterações serão comunicadas aos usuários através da plataforma ou por e-mail.
