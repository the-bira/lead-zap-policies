2. Descrição dos Serviços

O LeadZap é uma plataforma que oferece:

  • Atendimento via WhatsApp com chatbot de IA;
  • Qualificação de leads com base em interações e respostas a perguntas-alvo;
  • Agendamento automático de chamadas ou serviços;
  • Dashboard para que o dono do negócio visualize o histórico de conversas, métricas de lead score e outros dados operacionais.

3. Uso da Plataforma

Para Donos de Negócio: O processo de onboarding inclui cadastro, conexão do WhatsApp, preenchimento de informações comerciais e acesso ao dashboard para gerenciamento de clientes e conversas.

Para Clientes Finais: Os clientes interagem com o chatbot via WhatsApp, autorizando o envio de mensagens e a coleta de dados para qualificação e agendamento.

Integração com APIs: A plataforma integra a WhatsApp Cloud API para gerenciar conexões e mensagens, armazenando os dados (como tokens e identificadores) para associá-los ao respectivo cliente.
