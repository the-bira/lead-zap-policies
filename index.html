POLÍTICA DE PRIVACIDADE 1. Introdução A presente Política de Privacidade descreve como o LeadZap coleta, utiliza, armazena e protege as informações dos usuários (tanto os donos de negócios quanto os clientes finais) que utilizam nossa plataforma para gerenciamento de leads, agendamentos e comunicação via WhatsApp com chatbot de IA. 2. Dados Coletados 2.1. Dados de Usuários (Donos de Negócio) Dados cadastrais: Nome, e-mail, senha (armazenada de forma criptografada) e imagem do perfil. Dados de autenticação: Informações provenientes do Active Directory (quando aplicável), registros de login, e dados de sessão. Dados operacionais: Informações sobre configurações de conta, conexões com o WhatsApp, tokens e identificadores (como phone_number_id) obtidos via API do Meta, dados de agendamentos e histórico de conversas. 2.2. Dados dos Clientes Finais (Leads) Contato: Número de telefone, nome (quando fornecido) e quaisquer dados coletados durante a interação com o chatbot (respostas às perguntas-alvo, lead score, etc.). Histórico de Conversa: Mensagens enviadas e recebidas, registros de agendamentos (call ou serviço) e interações registradas na plataforma. 3. Como Utilizamos os Dados Para prestar os serviços: Utilizamos os dados para viabilizar a comunicação via WhatsApp, qualificar leads por meio de chatbot com IA, agendar serviços e exibir históricos e métricas no dashboard do usuário. Melhoria e Personalização: Os dados ajudam a otimizar a experiência do usuário, personalizar o atendimento e aprimorar os fluxos de mensagens. Segurança: Integramos nossos sistemas a ferramentas de segurança (como Active Directory e soluções de endpoint) para garantir o gerenciamento centralizado dos hosts e proteção da infraestrutura. 4. Compartilhamento e Armazenamento dos Dados Armazenamento: Os dados são armazenados em bancos de dados gerenciados (ex.: PostgreSQL) e estão sujeitos a políticas de segurança, backup e acesso restrito. Compartilhamento: Não compartilhamos os dados com terceiros, exceto quando necessário para a prestação do serviço (por exemplo, integração com a API do Meta para o WhatsApp) ou para cumprir obrigações legais. Token de acesso e Identificadores: Os tokens e identificadores gerados pela integração com a API do Meta serão armazenados de forma segura e associados ao respectivo cliente, permitindo o gerenciamento individualizado mesmo que a operação seja centralizada. 5. Direitos dos Usuários Acesso, Correção e Exclusão: Os usuários podem solicitar acesso, correção ou exclusão dos seus dados pessoais. Consentimento: Ao utilizar a plataforma, o usuário concorda com a coleta e uso dos dados conforme descrito nesta Política. Revogação de Consentimento: Os usuários podem revogar seu consentimento, porém isso poderá impactar a prestação dos serviços. 6. Segurança dos Dados Medidas Técnicas e Administrativas: Implementamos medidas de segurança (como criptografia, políticas de acesso restrito e auditoria de logs) para proteger os dados contra acesso não autorizado, divulgação ou destruição. Integração com AD e Ferramentas de Endpoint: Utilizamos o Active Directory para gerenciamento centralizado de usuários e políticas de segurança, assegurando que todos os hosts da rede estejam protegidos com softwares de segurança de endpoint e atualizações regulares. 7. Alterações na Política de Privacidade Esta política pode ser atualizada periodicamente. Quaisquer alterações serão comunicadas aos usuários através da plataforma ou por e-mail. 8.
